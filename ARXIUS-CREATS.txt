========================================
FUNCIONALITATS VISUALS
========================================

🎨 Diagrames SVG:
  - Taula de billar proporcional (2:1)
  - Boles: Blanca, Groga (#1), Vermella (#3)
  - Trajectòries amb colors diferents:
    * Verd: sense efecte
    * Blau: amb efecte
    * Vermell: alternativa
  - Sistema de 7 diamants (bandes llargues)
  - Sistema de 3 diamants (bandes curtes)
  - Efecte de brillantor a les boles
  - Vora de fusta realista
  - Gradient de tapet verd

🎬 Animacions:
  - Trajectòria completa animada
  - Dibuix progressiu del camí
  - Hover effects
  - Pulse al completar
  - Fade in/out

========================================
PRÒXIMS PASSOS
========================================

1. ⚠️ IMPORTANT: Genera les icones PNG
   - Obre: icons/icon-generator.html
   - Descarrega icon-192.png i icon-512.png
   - Col·loca-les a la carpeta /icons/

2. Prova l'aplicació:
   - Obre index.html en el navegador
   - O executa un servidor HTTP local

3. Instal·la com a PWA:
   - Cerca la icona d'instal·lació
   - Afegeix a pantalla d'inici

4. (Opcional) Desplega online:
   - GitHub Pages
   - Netlify
   - Vercel
